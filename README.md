Instalação
Copie a pasta Runaway Invite Tools para:
World of Warcraft\_retail_\Interface\AddOns\
Inicie o jogo e confirme que o addon está ativo em AddOns na tela de seleção de personagem.
Fila de Summon
O que acontece	Como funciona
Alguém digita 123 no chat de raid	Entra automaticamente na fila
A janela aparece com brilho dourado pulsante	Fica visível enquanto há alguém na fila
Ícone de classe + nome colorido	Identifica visualmente cada jogador
Cronômetro ao lado do nome	Mostra há quanto tempo o jogador está esperando
Clique no ✓ ao lado do nome	Marca como summonado e remove da fila
Fila chega a zero	Janela fecha automaticamente
A janela pode ser arrastada para qualquer posição — a posição é salva entre sessões.

Alerta para Warlocks
Quando um jogador entra na fila e você é Warlock, aparece um painel roxo pulsante no centro da tela com:

Nome do jogador que quer summon (colorido pela classe)
Quantidade de outros jogadores na fila
Ícone do Ritual of Summoning clicável → clique nele para já conjurar o spell
Som de alarme para chamar atenção
O alerta some automaticamente assim que você começar a conjurar o Ritual of Summoning.

Comandos
Comando	Ação
/rit	Mostra ou esconde a janela da fila
/rit clear	Limpa toda a fila manualmente
/rit test	Adiciona jogadores fictícios para testar
/rit help	Exibe ajuda no chat
Observações
Detecta 123 apenas no chat de Raid e Raid Leader — não funciona em outros canais.
Fora de raid, use /rit test para ver o addon em funcionamento.
O botão de conjura no alerta Warlock não funciona em combate (limitação do WoW para spells conjurados por botão).
